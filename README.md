# CLICKER BLAHAJ / OWL 


Feed a poor owl in need so he can grow stronger and be independant. Press Space to click to give him more food and with the upgrades you'll be able to give him even more food quicker so he'll get full.

This project was made for a OneKey project where I decided to do a silly game to entertain anyone who wants to just do nothing while doing something, it's composed by three websites where the start is the instructions and test page, the second web is the main game where you just click the owl to feed it and change between the owl and the upgrades to get more clicks quicker until you get to a million, the upgrades cost more each time you buy them, finally the third page is the congratulations page where you can finally leave or go back to the instruction page to start again. The three pages are joined so it will redirect you to the next one when you finish a certain number of clicks you'll get redirected to the next page. All click have soundbourds so you can be certain that its counting, the only thing that doesn't have them is the progress bars to change between the owl and upgrades. 

This was the first website I've ever made with code and first time programming, that's why It took so long to make.

This code was made with Html, javascript and Css.
